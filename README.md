# 🚀 Interview Copilot AI

An AI-powered chatbot designed to simulate real interview experiences across multiple domains like Frontend, Backend, HR, System Design, AI/ML, and DevOps.

---

## 🌟 Overview

Interview Copilot is a purpose-built chatbot that helps users practice interview questions in a structured and domain-specific way.

Instead of being a generic chatbot, it adapts its behavior based on the selected role, providing more relevant and focused responses.

---

## 🎯 Features

- 🧠 Domain-specific interview chatbot
- 💬 Real-time chat with AI (Groq API)
- ⚡ Fast and responsive UI
- 🎨 Animated tech-themed background
- 🧾 Markdown-supported responses (code blocks, lists, formatting)
- ⌨️ Enter-to-send message support
- 📱 Fully responsive design

---

## 🧩 Tech Stack

### Frontend
- Next.js (App Router)
- React.js
- Tailwind CSS

### AI Integration
- Groq API (LLM-based responses)

### UI & Animations
- Framer Motion
- Lucide React Icons
- Custom Canvas Animations

# 🚀 How to Run the Project (IMPORTANT)

Follow these steps carefully:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/interview-copilot-ai.git
```

---

### 2️⃣ Navigate into the Project

```bash
cd interview-copilot-ai
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

### 4️⃣ Add Environment Variables

Create a file named `.env.local` and add:

```env
GROQ_API_KEY=your_api_key_here
```

---

### 5️⃣ Start the Development Server

```bash
npm run dev
```

---

### 6️⃣ Open in Browser

```bash
http://localhost:3000
```
